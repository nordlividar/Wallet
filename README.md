# WorldPulse Wallet

A "living" wallet for the WorldPulse (WPU) token on Sepolia, with AI-powered spending suggestions.

## Tech Stack
- **Smart Contract:** Hardhat, Solidity
- **Frontend:** HTML, CSS, JavaScript, Ethers.js
- **AI Server:** Python, Flask
